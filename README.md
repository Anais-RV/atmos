#  Atmos - Sistema Meteorológico

Sistema fullstack para gestión y visualización de datos meteorológicos.

---

##  Inicio Rápido

```powershell
# 1. Clonar
git clone https://github.com/Anais-RV/atmos.git
cd atmos

# 2. Configurar todo automáticamente
.\run.ps1 setup

# 3. Iniciar backend (en esta terminal)
.\run.ps1 backend

# 4. Iniciar frontend (en otra terminal)
.\run.ps1 frontend
```

**¡Listo!**  
- Backend: http://127.0.0.1:8000  
- Frontend: http://localhost:5173
